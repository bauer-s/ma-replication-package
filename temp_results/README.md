# External Experiment Results (temp_results)

This folder is a local placeholder for the **raw experiment logs** that were originally stored in the whisker-cluster-experiments under `results/`.

## Notes

- The **raw log files** in this folder are ignored by git (see the repository `.gitignore`).
- If you do not have access to the original logs, you can still use the pre-generated CSVs in `results/raw_data/` and the figures/tables in `results/data/`.
